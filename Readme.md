# Convergence method

Obtener la raíz de una ecuación como la converencia de dos ecuaciones

$f(x) = a + b \rightarrow f(x) = f(a) - f(b)$

### Ejemplo de uso
```Octave
	Metodo de convergencia
f(x) = --> x^3-9x^2+x+(90/%pi)

...

fx1 = (1*x^3) + (-9*x^2)
fx2 = (1*x) + 28

f(x) = (1)x^3 + (-9)x^2 + (1)x + (28.64789)
x = 2.1135254 con 34 iteraciones
```
